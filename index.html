CSS

  • CSS (Cascading Style Sheets) is the code that styles the content of the page in web development
  • CSS is what you use in order to selectively customiuze the sstyle of HTML elements.
  • there are 2 types of CSS:
    • Inline css: consists of using the style attribute to change the color of a single h1 heading
    • External css: is preferred in most cases because it allows developers to keep all their css rules in a seperate file
    • when developers use external style sheet they link the file using a link element palced in the htmls head
  • Seperation of concerns is a widely used term in web development and refers to seperating code into defferent files based on use.
    • A margin indicates how much space we want around the outside of an element.
    • A padding indicates how much space we want around the content inside an element.
